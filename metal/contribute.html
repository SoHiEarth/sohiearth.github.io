Contribute to Metal

Metal is an open-source project, and contributions are welcome!

You can contribute in various ways:

  • Code contributions: Submit pull requests on GitHub.
  • Documentation improvements: Help us make the documentation better.
  • Bug reports: Report any issues you find.
  • Feature requests: Suggest new features or enhancements.
